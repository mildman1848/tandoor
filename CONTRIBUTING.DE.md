# Contributing

English: [CONTRIBUTING.md](CONTRIBUTING.md)

## Vor einem Pull Request

- Änderungen sollten sich auf das Image, den Compose-Stack oder die Automatisierung konzentrieren.
- Aktualisiere `README.md`, `README.DE.md` und die Changelog-Dateien bei Verhaltensänderungen.
- Prüfe Build, Startverhalten und datenbankbezogene Änderungen vor einem Release.
